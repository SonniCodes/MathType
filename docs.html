Introduction: What is Aleph

Welcome to Aleph, a LaTeX WYSIWYG editor based on Mathquill. Aleph is a powerful tool that allows users to easily create and edit LaTeX documents in real-time, without the need for any knowledge of LaTeX syntax.

Aleph provides a user-friendly interface that is intuitive and easy to use, allowing users to focus on their content rather than the technical aspects of LaTeX. With Aleph, you can create complex mathematical equations, tables, and diagrams with ease, and see the results in real-time as you work.

Aleph is built on top of Mathquill, a popular JavaScript library that provides a user-friendly interface for entering mathematical equations. By building on top of Mathquill, Aleph inherits many of its powerful features, such as support for LaTeX commands and symbols, as well as the ability to export LaTeX code.

Whether you're a seasoned LaTeX user or a beginner, Aleph is the perfect tool for creating high-quality documents quickly and easily. With its user-friendly interface and powerful features, Aleph is the future of LaTeX editing.

What is Formula and Formulate

Formula and Formulate are subsets of the Aleph project. This section introduces you to core concepts on each tool and when to use each tool.

Formula: The One-Minute TeX Editor

Formula is a subproject of Aleph that allows users to quickly create LaTeX documents without the need for technical expertise. Its user-friendly interface lets you jot down ideas, take notes, and create documents in real-time. Formula is fully developed and ready to use, making it an efficient tool for those who want to create LaTeX documents quickly and easily.

Formulate: Long Projects Made Fast

Formulate is a subproject of Aleph that is designed for longer, more complex LaTeX projects. Currently in development, it promises to offer a comprehensive set of features for creating tables, diagrams, and images with ease. Formulate will also provide advanced formatting options such as custom page layouts and styles. Stay tuned for updates on the development of Formulate!

Using Formula

To begin typing in Formula, simply click on the editor window and start typing. As Aleph is based on the Mathquill library, it uses a math-friendly syntax for inputting equations and symbols.

For example, to type a fraction, type the numerator, followed by a forward slash /, and then the denominator. To add a superscript, use the caret ^ symbol, and for a subscript, use the underscore _ symbol.

You can also use LaTeX commands directly by typing the backslash \ symbol followed by the command name. For example, typing \sqrt will create a square root symbol.

Aleph also includes a toolbar with buttons for commonly used symbols, making it easy to add them to your document with a single click. And with real-time rendering, you can see the results of your typing immediately as you type, making it easy to create complex equations and formulas quickly and accurately.

Enviornments

Formula also supports environments for creating LaTeX documents. To create an environment, simply type the backslash \ symbol followed by the environment name in square brackets. The environments in Formula imitate their LaTeX counterparts.

For instance, the gather environment allows you to write multi-line text in a centered format, while alignment environments, such as align, enable you to align equations along a chosen axis. By default, the keybinds to create an a newline is enter while ctrl+shift+enter creates a new column.

  • \gather: A multiline instance of math in one line. This environment is essential for Formulate multi-line math-content.
    • \begin{gather}b+b\\=2b\end{gather}
  • \array: Creates a gather with thin black border
    this is an Aleph specific command. This will not work with LaTeX
    • \begin{array}\text{Array}\ \ \end{array}
  • \align: Uses an invisible line to delimit where groups of text should be aligned.
    • \begin{align}2x+6&=30\\2x&=24\\x&=12\end{align}

Matrices

Similar to environments, matrices are also supported by typing \ and the corresponding matrix name. Unlike environments matrices use ctrl+enter to create a new column.

Below is a list of supported matrix environments:

  • \bmatrix: Creates a matrix with square brackets [ ].
    • \begin{bmatrix}1&0\\0&1\end{bmatrix}
  • \pmatrix: Creates a matrix with parentheses ( ).
    • \begin{pmatrix}1&0\\0&1\end{pmatrix}
  • \vmatrix: Creates a matrix with vertical bars | |.
    • \begin{vmatrix}1&0\\0&1\end{vmatrix}
  • \Vmatrix: Creates a matrix with double vertical bars || ||.
    • \begin{Vmatrix}1&0\\0&1\end{Vmatrix}
  • \matrix: Creates a matrix with no brackets or delimiters.
    • \begin{matrix}1&0\\0&1\end{matrix}
  • \amatrix: Creates an augmented b-matrix.
    Note that this is a custom command specific to Aleph. This will not work in LaTeX.
    • \begin{amatrix}1&0&x_1\\0&1&x_2\end{amatrix}

Nested Environments

While Formula supports a variety of LaTeX environments and matrices, it does not currently support nesting environments or matrices within one another. This means that you cannot have a matrix inside an environment or vice versa.

However, this feature is planned for the future and may be added in a future update. In the meantime, you can still create complex documents using the supported environments and matrices, while keeping in mind the current limitation of not being able to nest them within one another.

Large Operators

The following are a list on common large operators.

  • \int: Used to represent an integral.
    • \int_b^a\sin \left(x\right)dx
  • \oint: Used to represent a contour integral.
    • \oint_C\sin \left(x\right)dx
  • \dashint: Used to represent a Cauchy principal value integral.
    • \dashint_{-\infty }^{\infty }\frac{\sin ^2\left(z\right)}{z^2}dz
  • \sum: Used to represent a summation.
    • \sum_{\begin{gather}p:\ \text{prime}\\p\le n\end{gather}}^{}\frac{1}{p}
  • \prod: Used to represent a product.
    • \prod_{n=1}^{\infty }\left(1-\frac{1}{4n^2}\right)
  • \lim: Used to represent a limit.
    • \lim_{ x\to \infty }\frac{\cos \left(x\right)}{x}

To use these large operators in Aleph, simply type the backslash \ symbol followed by the operator name. For instance, to create an integral sign, type \int. Aleph will automatically format the operator to its corresponding LaTeX symbol, allowing you to create complex mathematical notation with ease.

You can tab into the limits of these large operators by using the left and right arrow keys or the up and down arrow keys.
